Git Branch

  • 8/1/2024
  • One-minute read

Several commands related to the git branch.

创建分支:

git branch test　＃以"test"分支为例子

切换分支

git checkout test

合并分支

如果你想要把test分支合并到master分支：

git checkout master #先切换回master分支
git merge test #合并操作

如果你想在合并后把原来的分支删除：

git branch -d test	#删除本地分支
git push origin --delete test	#删除远程分支
